Enlever le tag oups 01

Le tag oups a atteint son but. Enlevons-le et permettons aux commits qu’il référence d’être nettoyé.

Exécuter:

git tag -d oups
git hist --all

En sortie:

$ git tag -d oups
Deleted tag 'oups' (was 5cc6a21)
$ git hist --all
* 9da17c3 2012-05-06 | Ajout d'un commentaire (HEAD, v1, master) [Jim Weirich]
* 1f6ac8e 2012-05-06 | Ajouter une valeur par défaut (v1-beta) [Jim Weirich]
* ad21f9e 2012-05-06 | Utilisation de ARGV [Jim Weirich]
* c3df0aa 2012-05-06 | Mon premier commit [Jim Weirich]

Le tag oups n’est désormais plus listé dans le dépôt.
