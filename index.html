Git Immersion

Git Immersion est un tour guidé qui permet de voir les fondamentaux de Git, inspiré par la promesse que savoir une chose c'est l'expérimenter.

Git est un système très puissant et très élaboré pour la gestion de version de fichiers. Obtenir une compréhension de ses fonctionnalités ouvre aux développeurs une approche nouvelle et libératrice pour gérer son code source. Le plus sûr moyen de de maîtriser Git est de s'immerger dans ses utilitaires et ses différentes fonctions, pour expérimenter sa prise en main.
